Automatically add "Read More..." link for each blog post.

== Description ==
This plugin will put a "Read More..." link for each post of the blog page after the first image and the first paragraph. If no image present at the beginning, it will only show the first paragraph and the read more link after that. 

== Installation ==

1. Put read-more-link.php into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Replace the_content() with the_content_with_readmore_link() at blog post page. 

== Frequently Asked Questions ==

= How can I tell if it's working? =

Using that function at the blog post page, you will see a new "Read More..." link.
